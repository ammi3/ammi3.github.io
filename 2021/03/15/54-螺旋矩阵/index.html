54.螺旋矩阵

题目

给你一个 m 行 n 列的矩阵 matrix ，请按照 顺时针螺旋顺序 ，返回矩阵中的所有元素。

示例

提示

  1. m == matrix.length
  2. n == matrix[i].length
  3. 1 <= m, n <= 10
  4. -100 <= matrix[i][j] <= 100

题解

  1. 模拟循环
  2. 按照第一行–>最后一列–>最后一行–>第一列这种外层遍历，依次将二维数组“剥开”
  3. 这种剥开的方式，会使起始行r与终止行row逐渐靠近，同理列也是如此，故终止条件是起始行=终止行，起始列=终止列
