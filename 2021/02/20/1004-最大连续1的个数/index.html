1004.最大连续1的个数

题目

给定一个由若干 0 和 1 组成的数组 A，我们最多可以将 K 个值从 0 变成 1 。

返回仅包含 1 的最长（连续）子数组的长度。

示例

提示

  1. 1 <= A.length <= 20000
  2. 0 <= K <= A.length
  3. A[i] 为 0 或 1

题解

  1. 滑动窗口，每次移动right指针时判断当前值是否为1
    1. 若当前值为1，更新最大长度
    2. 若当前值为0，判断K的值，若K值大于0，则更新K值和最大长度；若K值为0，则更新左指针left
